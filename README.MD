### Render.com

1. Create a new Web Service
2. Connect your GitHub repository
3. Set environment variables
4. Add persistent disk at `/data`
5. Deploy

```yaml
# render.yaml
services:
  - type: web
    name: zent-launchpad
    env: node
    buildCommand: npm install
    startCommand: npm start
    envVars:
      - key: NODE_ENV
        value: production
      - key: RENDER
        value: true
    disk:
      name: data
      mountPath: /data
      sizeGB: 1
```
